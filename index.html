1. 등록 템플릿 선택 & 엑셀 업로드

먼저 등록할 그립 템플릿을 선택하세요. 셀러가 보유한 엑셀을 올리면 그 템플릿 컬럼 기준으로 매핑합니다. 클릭메이트 양식은 자동 인식되어 매핑 없이 호환 등록됩니다.

2. 필드 매칭

선택한 그립 템플릿의 컬럼에 연결할 셀러 엑셀 컬럼을 지정합니다. 컬럼명이 유사하면 자동 추천되며, 직접 선택/수정할 수 있습니다. 카테고리·상품정보고시 상품군은 1단계에서 미리 지정되어 여기서는 다루지 않습니다.

3. 등록 전 검증 & 수정

등록 전에 그립 검증 규칙을 전수 적용했습니다. 실패할 항목을 화면에서 바로 고치고, 기계적 오류는 원클릭 자동 교정으로 처리하세요. 등록 후 이메일을 기다릴 필요가 없습니다.

4. 등록

검증을 통과한 상품만 등록합니다. 한 건이 실패해도 전체가 막히지 않으며, 자동 추가된 조합은 판매 OFF로 등록됩니다.
